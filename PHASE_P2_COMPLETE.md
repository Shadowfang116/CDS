## What Was Added

### Pilot Reset Script (`pilot_reset.ps1`)
- **Purpose:** One-command setup for demo environment
- **Features:**
  - Optional `-KeepVolumes` flag to preserve data
  - Health check polling (waits for services to be ready)
  - Migration handling with error recovery
  - Demo ID extraction from seed output
  - Comprehensive summary with URLs and credentials

### Smoke Test Script (`smoke_test.ps1`)
- **Purpose:** Automated verification of core functionality
- **Tests:**
  1. Health check (deep endpoint)
  2. Dev login (OrgA and OrgB)
  3. Dashboard data loading (tenant isolation)
  4. Demo case/document ID extraction
  5. OCR status check
  6. Rules evaluation
  7. Export generation (discrepancy letter + bank pack)
  8. Audit log verification
- **Features:**
  - Graceful handling of edge cases (missing documents, already-done OCR)
  - Clear PASS/FAIL output
  - Exit code 0 on success, 1 on failure

### Enhanced Seed Script
- **Demo IDs:** Prints `DEMO_CASE_ID` and `DEMO_DOC_ID` for smoke tests
- **Demo Credentials:** Prints email/password for API testing
- **Rule Triggers:** Adds keywords like "encumbrance" and "mortgage" to OCR text
- **Idempotency:** Safe to re-run (upserts instead of duplicates)
