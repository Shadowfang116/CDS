# Example Real Documents (Safe Placeholders)

This folder contains **synthetic, safe-to-commit** PDFs that demonstrate the real-doc workflow structure.

These PDFs are generated via reportlab and contain no PII.

## Purpose

- Show the expected file naming convention
- Demonstrate folder structure
- Provide testable examples for CI/CD
- Safe to commit to repository

## Files

- `EXAMPLE_CASE__DHA_NDC__PLACEHOLDER.pdf` - Example DHA NDC document
- `EXAMPLE_CASE__SALE_DEED__PLACEHOLDER.pdf` - Example sale deed document
- `EXAMPLE_CASE__FARD__PLACEHOLDER.pdf` - Example fard document

## Usage

For actual UAT, use `docs/pilot_samples_real/` with real documents (gitignored).
