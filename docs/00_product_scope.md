# Use-Case Scope (MVP)

## Primary workflow
Punjab-first mortgage against society plot + urban house.

## Property types
- Society plot (MVP)
- Urban house/apartment (MVP)
- Rural/agri (Phase 2)

## Outputs
1. Executive Summary (risk + decision)
2. Exceptions list (issues)
3. CP list (requirements)
4. Drafts (DOCX): discrepancy letter, undertaking/indemnity, internal legal opinion skeleton
5. Annexures (evidence references by doc + page)

## Pass/Fail logic (MVP)
- PASS if: no High-risk exceptions open
- CONDITIONAL PASS if: high-risk allowed only with indemnity + approvals (must be explicitly recorded)
- FAIL if: high-risk unresolved and not waivable under policy
